ייצוא נתונים אישיים במהירות

כדי לייצא את הנתונים שלך:

  • היכנס לחשבונך באתר

  • לחץ על "ייצוא נתוני חיסונים"

  • בחר את פורמט הקובץ הרצוי (CSV או PDF)

  • לחץ על "ייצוא"

  • הקובץ שיוּצא יכיל את כל נתוני החיסונים שלך. תוכל להשתמש בקובץ זה כדי להוכיח את מצב החיסון שלך בעת נסיעה לחו"ל.
חשוב לציין:
ייתכן שחברות תעופה ומדינות מסוימות ידרישו תיעוד חיסונים נוסף.
ודא שאתה בודק את הדרישות הספציפיות של חברת התעופה ו/או המדינה אליה אתה נוסע לפני נסיעתך.
לשאלות נוספות, אנא צור קשר עם שירות הלקוחות שלנו.
